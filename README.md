# am colorschemes


### color schemes
 - amlight.vim
 - amdark.vim    
 - amcolors.vim:
   combines amlight and amdark
 - ampresent.vim:
   light vim colorscheme optimized for presentations
 - amblack.vim:
   dark vim colorscheme optimized for presentations

### plugin support:
 - CleverF         
 - HighlightedYank / Flashy         
 - indent line 
 - indent guides
 - braceless 
 - signify
 - ALE (Asynchronous Lint Engine)
 - GitGutter
 - multi-cursor
 - tagbar
 - CtrlP
 - poppy

### special file type support
 - xml
 - html
 - json
 - Mathematica (mma)
